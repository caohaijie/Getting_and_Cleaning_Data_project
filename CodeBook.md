# Getting and cleaning data - Course project - Code book
This code book describes the variables present in 'clean_data.txt' and 'clean_average.txt'.
## Identifiers (dimensions)
* 'activity' -  The activity that was being performed when the measurements were taken (walking, walking upstairs, walking downstairs, sitting, standing, laying)
* 'subject' - ID of the test subject who performed the activity (from 1 to 30)
